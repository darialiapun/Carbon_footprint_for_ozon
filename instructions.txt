# Проект: Ozon Eco Tracker (Chrome Extension)

## 1. Основная концепция
Расширение для Chrome, которое анализирует карточку товара на Ozon.ru и выводит:
- Углеродный след самого товара (через БД/API).
- Углеродный след упаковки (LDPE пластик), рассчитанный на основе габаритов.
- Суммарный экологический импакт в понятных сравнениях.

## 2. Технический стек
- Manifest V3.
- Climatiq API (для точных коэффициентов эмиссии).
- Хранение ключей в .env (через процесс сборки или константы).

## 3. Логика извлечения данных (Scraping)
- Отслеживание перехода на страницу /product/ через MutationObserver.
- Извлечение Названия (h1) и Категории (breadcrumbs).
- Извлечение характеристик: "Размеры, мм" (длина, ширина, высота) и "Вес товара, г".

## 4. Математическая модель упаковки
- Тип: Курьер-пакет LDPE (полиэтилен 04).
- Плотность LDPE: 920 кг/м³. Толщина стенки: 60 микрон (0.00006 м).
- Формула веса пластика: Масса (кг) = Площадь поверхности товара (2*(ab + bc + ac)) * Толщина * Плотность.
- Эмиссия упаковки: Масса пластика * 2.11 (кг CO2e/кг материала).

## 5. Интеграция с БД (Climatiq)
- Endpoint: https://api.climatiq.io/estimate/v1/general/custom
- При отсутствии данных в API или ошибке — использовать фоллбэк на константы:
  - Одежда: 7.5 кг CO2e/кг
  - Электроника: 80 кг CO2e/ед
  - Пластик: 6 кг CO2e/кг

## 6. Динамический UI Виджет (Финальный дизайн)
- **Местоположение:** Внедрять ТОЛЬКО в карточку товара Ozon (рядом с кнопкой покупки или ценой).
- **Визуальный эталон:** Использовать сгенерированную картинку (приложена к чату) как  референс для стилей image(1)
- **Компоненты:**
  1. **Заголовок:** Общий след (Товар + Упаковка + Логистика).
  2. **Круговой трекер:** Месячный лимит 166 кг CO2. Плавная смена цвета: Зеленый (<10%) -> Желтый (50%) -> Красный (>80%).
  3. **Эко-герой:** Блок со случайным животным из массива (имя животного + иконка + процент от его жизненного следа).
- **Стилизация:** Белый фон, Border-radius: 15px, Box-shadow: 0 4px 12px rgba(0,0,0,0.1).
